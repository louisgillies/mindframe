Animal Power

Seven animal-based cognitive modes

Rabbit

Multiply Ideas

Rabbits multiply fast. So should ideas. Generate many directions from a single starting point by shifting angle, audience, and format.

Prompt frame 
Take this idea and multiply it into 10 different variations. For each variation: change the angle, change the audience, change the format.

Owl

Deep Analysis

Owls are patient, sharp-eyed, and wise. Slow down to examine what others rush past -- hidden factors, overlooked perspectives, subtle patterns.

Prompt frame 
Think like an owl — slow, observant and analytical. Examine this problem from multiple perspectives and identify the hidden factors most people overlook.

Ant

Small Steps

Ants don't carry the whole load at once. Break massive undertakings into tiny, persistent steps -- the smallest realistic actions.

Prompt frame 
Think like an ant. Break this goal into the smallest possible steps someone could realistically complete.

Dolphin

Creative Solutions

Dolphins are curious, playful, and inventive. Encourage unconventional ideas that most people wouldn't consider.

Prompt frame 
Think like a dolphin — curious, playful and inventive. Generate creative solutions to this problem that most people wouldn't normally consider.

Eagle

Big Picture Strategy

Eagles soar high and see the entire landscape at once. Pull back from details to reveal how all the pieces connect.

Prompt frame 
Think like an eagle flying high above the landscape. Explain the long-term strategy behind this idea and how the pieces connect.

Beaver

Practical Systems

Beavers build dams -- complex, functional structures assembled methodically. Design practical systems with clear sequential steps.

Prompt frame 
Think like a beaver building a dam. Design a practical system that solves this problem step by step.

Elephant

Cross-Disciplinary

Elephants have powerful memories and deep social intelligence. Connect ideas to insights from entirely different fields.

Prompt frame 
Think like an elephant with a powerful memory. Connect this idea to insights from other fields such as psychology, economics, science or history.

How to Use

Three ways to activate animal thinking

01 Single Animal Mode

Specify an animal directly or describe what you need. Mindframe maps your intent to the right cognitive mode.

You say "Use rabbit mode on this brief"
Or just "I need lots of ideas for this"

02 Auto-Select Mode

Describe what you need and Mindframe picks the right animal for you.

You say Animal
"Give me lots of options"Rabbit
"Analyze this deeply"Owl
"Break this into steps"Ant
"Get creative with this"Dolphin
"Show me the big picture"Eagle
"Build me a system for this"Beaver
"Connect this to other fields"Elephant

03 Multi-Animal Mode

Combine animals for more powerful thinking. Use these recipes or invent your own.

Rabbit + Owl

Diverge then converge

Generate many ideas, then deeply analyze the best ones.

Eagle + Ant

Strategy to action

See the big picture, then break it into executable steps.

Dolphin + Beaver

Dream then build

Generate creative solutions, then design the practical system.

Elephant + Owl

Cross-pollinate then scrutinize

Draw from other fields, then rigorously analyze.

Rabbit + Owl + Ant + Dolphin + Eagle + Beaver + Elephant

Full Safari

Run all seven modes in sequence for the most thorough exploration of any problem.

Install

Get started in under a minute

Run this command in your terminal:

/install github.com/louisgillies/mindframe

Install globally via npm:

npm install -g mindframe

A Custom GPT version is coming soon.

In the meantime, check the setup guide in the README for instructions on using Mindframe prompts with ChatGPT.

Copy the prompt frames above into any LLM's system prompt or conversation. No installation required -- just paste and go.
